# 🗺️ Dhaka Interactive Map Viewer

A simple Python application that creates an interactive map of **Dhaka, Bangladesh** using the **Folium** library. The application automatically generates an HTML map file and opens it in your default web browser.

## 📸 Project Preview

The map displays:
- 📍 A marker pointing to Dhaka, Bangladesh
- 🔍 Zoom and navigation controls
- 🌍 Interactive OpenStreetMap view
- 🖱️ Pan and zoom functionality

---

## 🚀 Features

- 🗺️ Interactive digital map
- 📍 Custom location marker
- 🌐 Opens automatically in a web browser
- ⚡ Lightweight and easy to use
- 🐍 Built with Python and Folium
- 🌎 Uses OpenStreetMap tiles

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Dhaka-Map-Viewer.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd Dhaka-Map-Viewer
```

### 3️⃣ Install Required Library

```bash
pip install folium
```

---

## ▶️ How to Run

Run the Python file:

```bash
python main.py
```

After running the program:

- A file named `dhaka_map.html` will be created.
- The map will automatically open in your default browser.
- You can zoom, drag, and explore the map.

## 💻 Source Code Overview

The program:

1. Creates a map centered on Dhaka.
2. Sets the zoom level.
3. Adds a marker with a popup.
4. Saves the map as an HTML file.
5. Opens the map in the browser.
